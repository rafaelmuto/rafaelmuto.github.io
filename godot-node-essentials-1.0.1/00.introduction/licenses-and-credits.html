Licenses and credits

The source code and art assets in this project are respectively available under the MIT and CC-By licenses.

In short, for you, it means that you can reuse any code snippets or elements from the demo in your projects, even commercial ones. All you need is to include the MIT license. This is the same as with Godot itself.

If you reuse the sprites, textures, sounds, or animations bundled with the project, you need to credit us like so: [list assets used] CC-By 4.0 GDQuest
