How to download the course files

You can download the course to enjoy offline, DRM-free.

You can find the course files under every guide on this website, inside the folded “course downloads” section.

To download the files:

  1. Click the folded section below that says “Course files.”
  2. Once expanded, you can click any file to download it.

The course comes as zipped archives. To access the files, you need to extract them from the archives. Your system should come with a program that does that pre-installed.

To unzip the files:

  1. In your file explorer, navigate to the folder containing the downloaded file.
  2. Right-click the zip file and select the option saying “extract” (Windows, Linux) or “uncompress” (MacOS).

You should see a new directory named like the zip archive. It contains the course files.
